# Reservas internacionales brutas

Serie diaria oficial del BCRA, variable 1 de la API de estadísticas monetarias v4. Se publica en millones de USD desde el 3 de enero de 1996.

El estado es `official_provisional`: el BCRA advierte que las cifras están sujetas a cambios de valuación. Esta serie no representa reservas netas ni líquidas y no aplica deducciones por encajes, swaps, SEDESA, BOPREAL, repos u otros pasivos.

El pipeline conserva cada respuesta paginada con SHA-256, rechaza fechas duplicadas, valores no positivos, cambios de identificador y pérdida de cobertura histórica.
